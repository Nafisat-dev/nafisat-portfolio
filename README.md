## 🌟 About the Project

This portfolio is a comprehensive showcase of Nafisat Adepoju's skills and expertise in web development, with a focus on creating high-converting e-commerce experiences and impactful digital products. It serves as a central hub for potential clients and employers to explore projects, understand services, and get in touch.

---

## ✨ Key Features

- **Modern & Responsive Design**: A clean, professional, and fully responsive layout that looks great on all devices.
- **Interactive Project Showcase**: A filterable gallery of projects with detailed modal views, including image carousels, overviews, and tech stacks.
- **Dedicated Service Pages**: Detailed pages for various services like Shopify, React, and WordPress development.
- **Dynamic Animations**: Smooth animations and transitions powered by `framer-motion` and `gsap`, and scroll-based fade-in effects for a premium user experience.
- **Integrated Calendly Booking**: A "Book a Free Call" button that opens a Calendly modal for easy appointment scheduling.
- **SEO Optimized**: Includes comprehensive meta tags for better search engine visibility and social media sharing.
- **Comprehensive UI Component Library**: A rich set of custom UI components built with Radix UI and styled with Tailwind CSS.

## 🛠️ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- [Node.js](https://nodejs.org/en/) (version 18 or later recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/Nafisat-dev/nafisat-portfolio.git
    ```
2.  **Navigate to the project directory:**
    ```sh
    cd nafisat-portfolio
    ```
3.  **Install dependencies:**
    ```sh
    npm install
    ```
    or
    ```sh
    yarn install
    ```

### Running the Development Server

To run the app in development mode, use the following command:

```sh
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

### Building for Production

To create a production build, use the following command:

```sh
npm run build
```
This will create a `dist` folder with the optimized production build of your application.
